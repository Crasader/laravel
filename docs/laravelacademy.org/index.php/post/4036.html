[武汉]湖北鄂恒科技有限公司诚招PHP开发工程师及前端开发工程师[10-18k]

发布于 2016-05-05 2016-05-05 18:14:57 ehenghr 1005 Views

公司简介

Born to be proud

天生骄傲

8年！一段往前看很长远，而往后看一瞬间的时间。

2007年互联网媒体大肆兴起，我们做了《湖北散装水泥信息网》，4年的时间我们成为本行业点击率、转载率领先的门户网站，成为了首个与中国散装水泥推广发展协会达成战略合作的区域性行业门户网站。

2008 年我们推出了本行业全国第二套省级网上统计直报系统，在两年后，我们受中华人民共和国商务部、中国散装水泥推广发展协会的委托制作了“全国散装水泥‘三位 一体’网上智能直报系统”，2012年产品正式上线，截至今日我们是全国第三套推出、第一套使用面积最大的官方指定无纸化智能统计上报系统。

2010年“鄂恒科技车辆远程卫星监控系统”正式上线，两年后，我们又一跃成为湖北省商砼行业“车辆卫星监控系统”在线率第一的企业。

2012年“砼管家--鄂恒科技商砼企业管理系统”正式上线，经历了两年不断的迭代，我们成为行业首个可根据客户需要灵活拆分功能的企业管理系统。

同年，我们骄傲的推出了“砼监察--鄂恒科技商品混凝土行业电子政务平台”及“砼监事--鄂恒科技商品混凝土行业协会管理平台”。

2014年，我们又一次骄傲地实现了“砼管家”、“砼监察”以及“砼监事”与微信平台的对接，完成了全行业首个跨手机平台的移动应用。

2015年，再次让我们骄傲地推出了“E排车—鄂恒科技商品混凝土企业车辆智能排班系统”，全国首款基于云计算的商品混凝土企业生产调度应用，让我们再一次感受到“骄傲”。

8年的时间，我们从一个平均年龄20岁的企业到平均年龄28岁的企业，我们骄傲打破了“嘴上没毛办事不牢”的世俗！

8年的时间，不少亲朋好友在问“这个小小的行业，能有何作为？”

8年的时间，不少竞争对手在笑“这么一个小小的团队，能有何作为？”

8年的时间，无数的客户投来猜忌的目光；

8年的时间，无数的人不理解我们为何夜夜挑灯；

8年的时间，家人对我们不懈的支持。

8年的时间，我也在无数次的问自己为什么？

这一切的答案在2014年10月，才从罗永浩的演讲中找到——

天生骄傲！

公司坐标

运营中心：武汉市武昌区中北路津津花园B座1604-1605

招聘岗位

PHP工程师

月薪：10K~18K

#职责
系统的服务器后端设计和开发。

##要求
1.精通操作系统、网络编程、系统编程；
2.精通至少一门服务器端开发语言，包括但不限于C#/PHP/Nodejs；
3.熟悉REST服务或其它RPC框架的原理，熟练掌握常用Web框架；
4.熟练使用主流关系数据库，熟悉SQL Server/Mysql/Postgres的任一种；熟悉常用NOSQL，包括mongodb、redis等；
5.熟悉高性能服务器开发，熟悉常见互联网基础架构，有大型互联网产品开发经验者优先；
6.良好的沟通能力，团队合作精神，有责任感；
7.对开源技术有强烈的兴趣和爱好，参与或向开发者提交过 bug 和 patch 者优先。

前端高级开发工程师

月薪：10K~18K

#职责

Web/Hybrid App以及复杂用户界面与交互应用的设计和开发。

##要求

1.精通html5/css/javascript；

2.熟悉主流前端框架；

3.能够写出高性能、可复用的前端组件；

4.至少了解一门服务器端开发语言，如C#/Java/PHP/Nodejs等；

5.有Hybrid App开发经验者优先；

6.良好的沟通能力，团队合作精神，有责任感；

7.对开源技术有强烈的兴趣和爱好，参与或向开发者提交过 bug 和 patch 者优先。
