[上海]上海比冲科技诚招PHP开发工程师[6-12k]

发布于 2016-04-14 2016-04-14 23:45:11 学院君 1204 Views

公司简介

上海比冲科技有限公司，与加拿大知名智能标识企业合作，给中国彩迷和球迷带来全新的购彩体验。创始人有丰富的互联网公司运营经验，曾涉足手游、餐饮行业。产品经理曾任国外知名电商产品总监。 中国创业实训基地孵化企业，目前已搭好舞台，只等长袖善舞的你来、来、来！

公司坐标

上海市杨浦区国定东路200号1号楼1108室

招聘岗位

PHP工程师

月薪：8000~12000

岗位职责：

  • 负责公司产品主要技术研究和接口设计；
  • 架构设计，搭建框架以及核心代码的编写;
  • 制定开发规范，控制产品开发质量;
  • 协助产品经理完成产品的验收和上线。

职位要求：

  • 熟练掌握LAMP，了解数据库、缓存等性能优化，3年以上项目开发经验；
  • 熟悉Laravel4或Laravel5框架；
  • 有良好的代码习惯，结构清晰，命名规范，代码冗余率低；
  • 熟悉PHP，深入理解PHP面向对象编程设计理念；
  • 熟悉HTML、DIV+CSS、Javascript、Ajax技术，熟悉MUI框架；
  • 熟悉MYSQL的开发、配置、维护、性能优化；
  • 能根据产品上线目标制订的迭代模型进行敏捷开发；
  • 较好的理解沟通能力，工作积极主动，责任心强，技术能独当一面者佳。

PHP程序员

月薪：6000~8000

岗位职责：

  • 负责公司产品前后台程序设计和开发；
  • 按项目计划完成编码编写、内部测试等工作；
  • 负责产品的后期维护与升级工作；
  • 根据产品需求，参与讨论技术解决方案。

职位要求：

  • 精通PHP编程，熟悉LAMP或WAMP架构, 1年以上项目开发经验；
  • 掌握HTML、DIV+CSS、Javascript、Ajax技术；
  • 拥有良好的代码习惯，结构清晰，命名规范，逻辑性较强，代码冗余率低；
  • 工作主动性强，有创新精神，能够承受一定的工作压力；
  • 良好的团队合作能力，执行能力与沟通能力佳。

公司福利

  1. 五险一金
  2. 交通方便
  3. 团队活动
  4. 氛围轻松
  5. 装备高新

公司愿景

提供通过便捷的途径带来娱乐和公益的彩票。
