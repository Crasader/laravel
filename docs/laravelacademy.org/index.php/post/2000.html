Laravel Homestead 新版本升级到 MySQL 5.7 和 Node.js 5.0

发布于 2015-11-15 2015-11-15 22:36:21 学院君 1737 Views

Homestead Vagrant Box 又升级了，新版本中升级 MySQL 到 5.7，同时升级到 Node.js 到 5.0。可以在本地主机中使用如下命令升级 Homestead：

homestead update

如果在此之前 Homestead 还在运行中，要先运行如下命令关闭 Homestead ：

homestead halt

升级要耗费一些时间，这取决你的网络连接速度，请耐心等待。

如果安装过程报错：

HTTP server doesn't seem to support byte ranges. Cannot resume.

对应解决办法是：进入~/.vagrant.d/tmp目录，删除该目录下的临时文件再重新运行homestead update命令。

此外，由于 MySQL 5.7 支持 JSON 字段类型（详情参考MySQL官方文档），即将发布的 Laravel 5.2 也将提供这一支持。
