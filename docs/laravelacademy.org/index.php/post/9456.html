惊了！竟然刷到了百度第一

发布于 2018-02-28 2018-07-26 10:10:16 SadCreeper 1124 Views

好吧，我承认我可能标题党了🤣

其实就是最近我发现我的昵称 sadcreeper 这个关键词在百度上搜索可以精准定位到我的博客了..

所以说值得开个贴纪念一下，顺便说说我是怎么做到的吧。。其实就是一点点 SEO 的知识，我没有系统的学过 SEO 优化，SEO 大神们肯定知道这都是小意思。。我就讲（sui）点（bian）小（luan）技（che）巧（dian）吧😏

首先，我觉得最重要的一点，就是要选一个偏僻点的关键词，如果你起个网名叫 faker 的话，估计不管你怎么 SEO 优化，搜索 faker 搜到的都会是韩国某电竞选手（社会我李哥）而不是你🤣，所以在你选好一个关键词之后，可以先上百度搜索一下，如果第一页出现的都是些乱七八糟的东西，说明这个关键词还没被占，就可以开始刷了..

然后最好建一个站，目标就是先要被百度收录（被百度爬虫采集到），可以尝试下列方法：

  • 用一些比较常见的域名后缀，比如com,cn之类的
  • 首页的<title>标签一定要包含你想刷的关键词，除了<title>之外，description 和 keywords 都把关键词加上
  • 除了上述位置之外，最好网站的每个页面都给他加上关键词，总之就是到处加，而且尽量往前面加，但是也不要影响到用户体验，毕竟这才是最重要的
  • 尽量少用 JS 生成代码，比如现在流行的 vue，react，至少关键内容要用原生的
  • 在其他网站上留你的站点链接，越多越好（很重要，尤其是那种大流量网站），交换友情链接也是经常用的套路
  • 尽量优化网站速度

坚持一段时间之后，就可以尝试搜索你的关键词，看看有没有你的站，没有就多往后翻翻，只要能翻到就成功了一半了，能翻到就说明被百度收录了，然后就继续刷吧，提升你的网站流量，再更多的地方留下你的网站链接，相信你的网站排名会越来越靠前的🤠

如果没有网站或者懒的建又想占一个关键词怎么办？那就找一个比较成熟的网站，用关键词做用户名建一个账号，然后多发文章动态之类的，就是要活跃起来，也是有用的，比较经典的例子就是如果你的 github 账号比较活跃，你就能在 google 上搜到：
