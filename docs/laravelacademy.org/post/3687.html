[杭州]靠谱创业团队诚招PHP开发工程师[10-20k]

发布于 2016-03-21 2016-03-21 22:51:23 学院君 2029 Views

公司名称

杭州津美其祐网络科技有限公司

公司坐标

浙江省杭州市滨江区滨安路1181号3号楼3层（华业科技园对面）

岗位职责

  1. 负责现有网站平台相关业务模块的分析、设计与维护。
  2. 参于各项目中的编码工作。
  3. 根据公司新的业务要求，开发相应的功能模块。

任职要求

  1. 有两年以上PHP开发经验，熟悉常用框架 Laravel 优先考虑，熟悉 Linux 系统
  2. 精通 PHP+MySQL 程序设计及开发，拥有良好的代码习惯，要求结构清晰，命名规范，逻辑性强，代码冗余率低。
  3. 熟悉面向对象的软件设计方法，对面向对象的设计模式有较深的理解并能熟练应用。

公司福利

  1. 14薪，五险一金，薪资多样，随便你怎么算
  2. 活动区比办公区大，还有1000平阳光大阳台，随便在哪里办公
  3. 每月1~2次的团建活动，随便你怎么玩
  4. 可乐、芬达和雪碧，随便你怎么喝
  5. 可以自带宠物，随便它怎么跑

薪资范围

10~20k，14薪

公司愿景

以酷的方式，酷的技术和酷的产品打造全国最大的儿童用品网上商城！

办公环境

简历投递
