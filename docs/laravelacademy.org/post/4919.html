公司简介

tech2ipo创见，中国最早的创投媒体之一。网址为 tech2ipo.com

TECH2IPO/创见成立于 2010 年，是国内首家专注科技创新创业的媒体平台。

我们通过对商业模式、新产品、新趋势和天使投资的解读，为科技创业注入源源不断的新动力。自成立以来，累计读者超过 5 亿人次。过去 3 个月，月平均综合阅读量超过 1300 万，影响力年增速 150%。

截止 2015 年年底，TECH2IPO/创见已经累计报道过 3351 个国内外创业项目。

曾参与政府部门多份创投行业报告撰写，并于去年出版创投领域畅销书籍《创业百道》并联合开课吧推出创业线上课程，在线观看量超过百万。

目前，除创见主站点外，我们还开设了以下垂直内容子站：

  • 科技尝新-天下没有难吃的科技
  • 土法创业-专注于说人话的创业媒体
  • 投资经理笔记-“偏见”是最具价值的投资
  • VR之家-你应该知道的关于 VR 的一切
  • 更多垂直子站正等待你与我们共同建设。

公司地点

中关村创业大街10号楼

公司福利

leader和PM都很萌，水果点心，餐补，交通补助，通讯补助，不定期聚餐，团建 有期权，虽然可能没什么卵用，但是肯定会给

岗位名称

PHP全栈工程师

薪资

15K~25K，能力卓越者上不封顶

岗位职责

  1. 负责公司科技媒体 PC站、M站的开发；
  2. 根据产品需求，完成功能开发、维护工作；
  3. 相关开发文档编写

任职要求

  1. 一年及以上PHP开发经验，掌握 LNMP 技术体系，熟悉 Laravel 开发框架更佳；
  2. 良好的数据库设计和优化能力，熟悉 MySQL，且掌握 redis、 MongoDB 等相关 NoSQL 技术更佳；且有大型网站开发经验，具有移动WEB开发经验更佳；
  3. 精通 JavaScript/CSS3/HTML5， 掌握最新特性及基本原理，并熟练应用；
  4. 熟悉前端技术，能够编写规范、可维护的前端代码，熟悉前端性能优化，掌握至少1种开发框架（AngularJS、Vue.js、Avalon.js、React等）；
  5. 熟悉 Linux 或 *nix 开发环境部署，熟练使用常用命令行工具及使用 Git 进行版本管理；
  6. 具有良好的技术视野，不局限于特定细分岗位，不局限于特定开发语言，不局限于特定工作流；

PS：不要求有大公司工作经验，但是要懂全栈工程师的概念。

还招实习生

  • 要求和上面差不多，但是实习生的话各项要求都会有所放宽；
  • 薪资为6K~8K，如果能力卓越，不是不可以再加，其他福利和全职一样

联系方式

产品经理邮箱：geyi@pe.vc
