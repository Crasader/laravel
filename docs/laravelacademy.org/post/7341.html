基于 Laravel 实现的类似 Reddit 的实时社交论坛系统 —— Voten

发布于 2017-06-18 2018-03-21 11:46:40 学院君 9964 Views

关于 Voten

Voten 是一个现代的实时社交书签系统，基于 Laravel 5.4 + Vue.js 开发，类似于 Reddit，但是更加美观，它实时、简单并且支持自定义。

  • 网址：https://voten.co/
  • GitHub：https://github.com/voten-co/voten

代码风格

Voten 遵循 PSR-2 代码标准和 PSR-4 自动加载标准，并且使用了 StyleCI 自动合并代码风格修复，所以基本不用担心代码风格问题。

软件栈

Voten 是一个 Laravel 应用，依赖以下软件才能运行：

  • Ubuntu 16.04.2 LTS
  • Nginx 1.10+
  • MySQL 5.7+
  • PHP 7.1+
  • Redis 3.0+
  • Git 2.8.4+
  • Pusher
  • Algolia Search

要想在服务器上安装所有这些软件，推荐使用自动安装服务，如 Codepier。

应用安装步骤

将 GitHub 仓库代码克隆到本地后，首先在项目目录下创建一个 .env 文件：

cp .env.example .env

按照本地环境修改 .env 中的相应配置信息，然后运行以下命令进行安装：

composer install
php artisan key:generate
php artisan migrate
php artisan passport:install
npm install
npm run production

创建后台用户

在根目录下运行如下命令：

php artisan db:seed --class=AdminUserSeeder

后台访问路由是 /backend，登录用户信息是admin / password。访问之前需要清除 Redis 缓存。

更多信息，请关注 Voten GitHub 仓库：https://github.com/voten-co/voten。
